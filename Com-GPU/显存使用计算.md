计算公式:[显存](显存.md)
主要是训练阶段占用显存较高，只计算训练阶段

总显存计算公式：[总显存 = 静态显存 + 动态显存](显存.md#^11cf60)

静态显存计算公式： [静态显存 = 权重 + 梯度 + 优化器](显存.md#^16134a)

假设这里模型的参数量为 21.7M（约2167万个参数），一个参数占用两个字节
权重 = 梯度 = 21,7 x 2 = 53.4 MB

优化器权重计算公式：[优化器 = 权重 + 一阶动量 + 二阶动量](显存.md#^b6e9dc)
Adam是使用32位来存，所以是 4 字节
优化器 = 53.4 x 2 x 3 = 320.4 MB

静态显存 = 权重 + 梯度 + 优化器 = 53.4 + 53.4 + 320.4 = 427.2 MB
